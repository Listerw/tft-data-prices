# TFT Data Prices

These are the data crunched from our server and generate outputs to TFT pricing channels. Most of the data are only available exclusively from TFT such as
- harvest craft
- service craft
- heist contracts
- other prices in **bulk**
- etc


### Folder Explanation

- `lsc` - League Softcore
- `lhc` - League Hardcore
- `std` - Standard Softcore
